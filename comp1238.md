## Week 1 - Information Storage
- Cuneiform was a very early writing from from mesopotamia around 3000BC
- Many modern terms such as typeface and font come from the history of book printing

## Week 2 - Text Representation
- Bit is the tiniest piece of information and holds either a 1 or 0
- Bytes are a group of 8 bits and often represent some ascii character

## Week 3 - Text Structure
- Markup language can be used to provide a computer instructions to display and interpret text
- Markdown and HTML are popular examples of markup languages

## Week 4 - Manipulating Text
- Text editing software can come in several forms (word processor, text editor, IDE, text-mode editor)
- RegEx is a language used for creating search patterns

## Week 5 - CLI
- CLI is a way of interacting with the computer using text-based commands
- Terminal is an application where you can type those commands

## Week 6 - Versioning
- Git is a distribured version control system
- GitHub is a platform used to host code repositories being tracked with git
